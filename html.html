Que es HTML

Siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje de marcado predominante para la elaboración depáginas web. El HTML se escribe en forma de «etiquetas», rodeadas por corchetes angulares (<,>) .

Que necesito para aprender HTML

Aparte de Dedicacion y empeño en lograr algo realmente bueno, necesitas un editor de HTML y por su puesto muchos recursos graficos, para que nuestro proyecto sea bueno.

Y pues el codigo podemos conseguirlo con tutoriales on-line, pdf y cursos sencillisismo o para mayor facilidad, usando editores codigo-visuales, como por ejemplo Dreamweaver que inserta el codigo mientras hacemos la parte grafica..

Porque aprender código HTML si hay editores que traen lo traen incorporado?

Muy simple, comprender el codigo HTML es de gran importancia ya que en alguna circunstancia podemos necesitar decodificar una pagina que nos haya gustado, o para insertar algun recurso, dentro de una parte del codigo, para hacer un llamado de algun Script o CCS.

Y porque seria muy tedioso dejar que la maquina haga todo por nosotros, el chiste es encontrarle algo de entusiasmo y errar para aprender.
