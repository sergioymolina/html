Listas de Menu

En HTML los menus no suelen tener muy buena apariencia pero para lograr hacer un menu agradable a la vista, hay que aprender a usar las etiquetas que lo componen. En este Caso, en HTML se usan las etiquetas, UL que se usa para abrir el menu o un submenu del menu. Tambien se usa la etiqueta LI que se usa para abrir los items de un menu o un submenu
